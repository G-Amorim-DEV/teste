HTML (HyperText Markup Language) é a linguagem padrão utilizada para criar páginas na web. Ele define a estrutura e o conteúdo de um site por meio de "tags", que organizam textos, imagens, links e outros elementos. O HTML é interpretado pelos navegadores, permitindo que as informações sejam exibidas de forma visual para os usuários.

O HTML é uma linguagem de marcação, o que significa que ele usa tags para estruturar o conteúdo. Com essas tags é possivel a inserção de arquivos como:

  • Textos
  • Imagens
  • Vídeos
  • Entre outros
